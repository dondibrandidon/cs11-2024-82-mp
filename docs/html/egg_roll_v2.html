tilt(degree: str, moves_left: int) → tuple[list[str], int, tuple[str, ...], bool]¶

The main logical interaction with the player! Simulates a tilting of the board, which moves the eggs.

Parameters:

  • degree (str) – A character in “fbrl” which determines direction.

  • moves_left (int) – Moves :class: Player has left, used to calculate points.

Raises:

ValueError – Raised when :param: degree is not in “fbrl.”

Returns:

Returns debugging logs, points gained or lost, array for grid animation, and “are there any eggs left?”

Return type:

tuple[list[str], int, tuple[str], bool]

class egg_roll_v2.Player(level_file: TextIOWrapper, freedom: dict[str, str] = {'backward': 'bB', 'forward': 'fF', 'leftward': 'lL,', 'rightward': 'rR'}, is_debug: bool = False)¶

Bases: object

This class is the controller and view handling all inputs and prints.

Variables:

  • name_to_char – Takes in word directions and returns its character

  • char_to_name – Takes in character directions and returns its word

  • char_to_symbol – Takes in character directions and returns its symbol

  • degrees – This turns the player input into the characters Level.tilt can process

  • valids – String containing all valid inputs from degrees

  • file_name – The name of the level_file.in

  • moves_left – Decrements with each successful character or “undo” input. treated as infinite if equal to -1.

  • max_moves – The number of moves as in level_file.in

  • current_level – The main Level file interacted with

  • points – All points gained by so far

  • game_end – The Game Over screen is displayed once True

  • past_moves – Array of symbols to denote past successful inputs

  • past_levels – Array of past Level states for undo inputs

  • past_points – Array of past points for undo inputs

  • current_input – The input for processing to Level.tilt

  • debug – Enables printing of debug info and disables time.sleep if True
