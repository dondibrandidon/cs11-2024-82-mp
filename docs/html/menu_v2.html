menu_v2 (Main Menu Module)¶

class menu_v2.Menu(is_debug: bool = False)¶

Bases: object

This is the class handling all menu displays, and highscore functionalities.

Parameters:

is_debug (bool, optional) – Makes debugging more convenient if True, defaults to False

Variables:

  • debug (bool) – Enables printing of debug info and disables time.sleep() if True

  • level_list (tuple[str]) – Array containing the names of all .in files in the .|levels folder when it was last checked

  • freedom (dict[str, str]) – Returns set valid characters for each word direction

directions: tuple[str, str, str, str] = ('forward', 'backward', 'rightward', 'leftward')¶

_generate_blank_highscore_files() → None¶

Generates a _score_level_file.txt for each level in level_list.

The format of a highscore file is as follows:

player_score1: int
player_score_datetime1: datetime.datetime
player_score_name1: str
...
player_score5: int
player_score_datetime5: datetime.datetime
player_score_name5: str

Note that only the TOP 5 mayors are kept in memory.
